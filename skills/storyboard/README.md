# storyboard

Generate a detailed video storyboard and script for any OpoClaw video production.

## Triggers

Use for: "haz un storyboard", "escribe el script del video", "planea el video", "story board"

## What to produce

1. **Hook** (0-5s): Opening statement that grabs attention
2. **Problem** (5-15s): What problem OpoClaw solves
3. **Solution** (15-40s): How OpoClaw works — the system, the agents, the dashboard
4. **Proof** (40-60s): Real results, numbers, capabilities
5. **CTA** (60-90s): Call to action

For each scene:
- Scene number
- Duration (seconds)
- Visual description (what's on screen)
- Narration text (exact words spoken)
- Mood/tone notes

## Format

Output as a structured document with all scenes timed and described in detail.
